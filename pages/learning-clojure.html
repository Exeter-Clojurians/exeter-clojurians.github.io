Learning Clojure

To get started you will want make sure you have a JVM. Then install Leiningen.

Then you need to get a decent editor (never use notepad). If you use Emacs you need to install CIDER and Paredit otherwise Cursive comes highly recommended . You can use Vim as well if you want - you will probably want to use Fireplace.

Then learn a bit of Clojure with http://learnxinyminutes.com/docs/clojure/ then Clojure for the Brave and True.

As you learn, solve a few puzzles.

The best place to look up documentation is Clojure docs.

When you are ready to do some web dev, by far the easiest way is to use the Chestnut template. This generates a project with client and server all set up and has all the plugins set up right so you have a repl to both the server and the browser – it can be quite frustrating trying to get it all set up by hand.
